French public services scraping

Project Info:

We studied french public services websites, apps, e-mails to find if trackers are present. We also organised data to do stats about it. We published a research paper about this project.

It uses python scripts to get public services apps from the playstore with keywords. We verify manually to check if it can be qualified or not as a public service. We run then Exodus Privacy and put all the results in CSV, in order to treat it.

Project Details:

  • Client:CITI Lab
  • Industry:Research
  • Technologies:Python, Exodus-Privacy's repo
  • Date:June 2020
